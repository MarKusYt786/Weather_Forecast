## ğŸ“¦ Installation
To get started with the Weather App, follow these steps:
### Prerequisites
* A web browser (e.g., Google Chrome, Mozilla Firefox)
* An internet connection
* An API key from OpenWeatherMap (optional, but recommended for heavy usage)

### Installation
1. Clone the repository using `git clone https://github.com/your-username/weather-app.git`
2. Open the `index.html` file in a web browser
3. If desired, replace the `apiKey` variable in `script.js` with your own OpenWeatherMap API key

### Running Locally
1. Open the `index.html` file in a web browser
2. Enter a city name in the input field and click the "Get Weather" button
3. The application will retrieve the weather data from the OpenWeatherMap API and update the user interface
